# 🛍️ Style Vibe – Full Stack E-commerce Store

## 📌 Description
Style Vibe is a full-stack e-commerce web application designed to manage and sell 100+ products across multiple categories. Users can explore products, search efficiently, manage their cart, place orders, and track their purchase history.

The platform also includes a powerful admin panel for managing products and orders. The application is fully responsive and works seamlessly across all devices.

---

## 🚀 Features

### 👤 User Features
- Browse products by category  
- Advanced product search with typo correction  
- View detailed product information  
- Add / remove items from cart  
- Place orders with:
  - Online payment (Razorpay)
  - Cash on Delivery (COD)  
- Track orders  
- View order history  
- Return products  

---

### 🔐 Authentication & Security
- JWT-based authentication  
- Protected routes using middleware  
- Secure environment variables with dotenv  
- CORS enabled to block unauthorized requests  

---

### 🛠️ Admin Features
- Add, edit, and delete products  
- Upload product images  
- Manage orders and view order details  

---

## 🧑‍💻 Tech Stack

### Frontend
- React.js  
- Tailwind CSS  
- React Query (data fetching & caching)  
- React Context API (state management)  
- React Router (navigation)  
- Axios (API calls)  
- Fuse.js (improved search with typo handling)  

---

### Backend
- Node.js  
- Express.js  

---

### Database
- MongoDB (MongoDB Atlas)  
- Mongoose (schema modeling)  

---

### ☁️ Deployment & DevOps
- Frontend hosted on Vercel  
- Backend hosted on AWS EC2  
- Images hosted on Cloudinary  
- CI/CD using GitHub Actions  
- Nginx for server configuration & SSL
